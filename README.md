## ゲーム設定

- 称号集め，レーティングシステム
- `The Proguesst`は１つの目標，その奥の隠し称号などがやり込み要素
- 強力なキャラクターゲーム＋ストーリーゲーム

- デイリーアリーナ（通称アリーナ）
  - 1日1回，決められたタイミングで同時開催される
- ブレイントレーニングジム（通称ジム）
  - いつでも遊べる
